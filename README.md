# Mon Vieux Grimoire – Projet OpenClassrooms

Ce projet est réalisé dans le cadre du parcours **Développeur Web** chez OpenClassrooms.  
L’objectif est de développer une **API sécurisée** pour une application de notation de livres, avec une intégration à un front-end React.

---

## Fonctionnalités

- Authentification sécurisée des utilisateurs
- Gestion des utilisateurs (inscription / connexion)
- Gestion des livres :
  - Ajout, modification et suppression de livres
  - Téléversement et stockage optimisé des images
  - Consultation de la liste et du détail des livres
- Système de notation des livres
- Protection des routes sensibles
- Architecture back-end basée sur le modèle MVC
- Communication avec un front-end React

---

## Technologies utilisées

- **Node.js**
- **Express.js**
- **MongoDB**
- **Mongoose**
- **JWT (JSON Web Token)**
- **Multer** (gestion des images)
- **React**
- **Vite** (environnement de développement)
